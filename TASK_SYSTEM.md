# TaskFlow - 태스크 관리 시스템

## 📋 태스크란?

**태스크(Task)**는 할 일, 작업, 업무를 의미하는 개별 작업 단위입니다.

### 예시
- "이메일 확인하기"
- "회의 준비 자료 만들기"
- "코드 리뷰하기"
- "장보기 목록 작성"

## 🎯 태스크 기능

### 1. 기본 정보
- **제목**: 태스크의 핵심 내용
- **설명**: 상세한 작업 내용 (선택사항)
- **우선순위**: 낮음/보통/높음
- **마감일**: 완료해야 할 날짜 (선택사항)
- **폴더**: 태스크를 분류할 폴더 (선택사항)
- **완료 상태**: 체크박스로 완료/미완료 표시

### 2. CRUD 기능
- ✅ **생성**: `/tasks/new` - 새 태스크 만들기
- ✅ **조회**: `/tasks` - 태스크 목록 보기
- ✅ **수정**: `/tasks/$id/edit` - 태스크 편집
- ✅ **삭제**: `/tasks/$id/delete` - 태스크 삭제
- ✅ **토글**: `/tasks/$id/toggle` - 완료 상태 변경

### 3. 통계 기능
- 전체 태스크 수
- 완료된 태스크 수
- 대기 중인 태스크 수
- 긴급 태스크 수

## 🎨 사용자 경험

### 1. 우선순위 표시
- **낮음**: 회색 (시계 아이콘)
- **보통**: 노란색 (경고 아이콘)
- **높음**: 빨간색 (경고 아이콘)

### 2. 마감일 관리
- 지연된 태스크: 빨간색 배경
- 마감일 표시: "YYYY-MM-DD" 형식
- 지연 표시: "(지연)" 텍스트 추가

### 3. 완료 상태
- 완료된 태스크: 취소선, 투명도 감소
- 미완료 태스크: 정상 표시

## 📁 폴더 연동

### 1. 폴더 분류
- 태스크를 폴더에 할당 가능
- 폴더별로 태스크 관리
- 폴더명이 태스크 카드에 표시

### 2. 폴더 선택
- 태스크 생성/수정 시 폴더 선택
- "폴더 없음" 옵션 제공

## 🔔 알림 시스템

### 1. 토스트 알림
- 태스크 생성 성공
- 태스크 수정 성공
- 태스크 삭제 성공
- 오류 발생 시 알림

### 2. 확인 모달
- 태스크 삭제 시 확인 창
- 실수 방지

## 📱 네비게이션

### 1. 대시보드 연동
- 빠른 작업: "새 태스크", "태스크 목록"
- 통계 표시: 태스크 현황

### 2. 태스크 목록 페이지
- 헤더: 뒤로가기, 새 태스크 버튼
- 통계 카드: 4개 지표
- 태스크 목록: 카드 형태

## 🎯 실제 사용 예시

### 개발자 워크플로우
```
09:00 □ "로그인 기능 코드 리뷰" (우선순위: 높음)
10:30 □ "API 응답 타입 정의" (우선순위: 보통)
14:00 □ "테스트 코드 작성" (우선순위: 낮음)
16:00 □ "문서 업데이트" (우선순위: 낮음)
```

### 완료된 태스크
```
✅ "인증 시스템 구현" (완료)
✅ "데이터베이스 스키마 설계" (완료)
✅ "UI 컴포넌트 개발" (완료)
```
